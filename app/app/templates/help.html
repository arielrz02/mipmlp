MIPMLP is preprocessing tool for 16S values.

The input file should contain detailed unormalized OTU/Feature values as a biom table, and the appopriate taxonomy as a tsv file,

and a possible tag file, with the class of each sample. See Example page for the formal of the input files.

The tag file is not used for the preprocessing, but is used to provide some statistics on the relation between the features and the class.

One can run MIPMLP even without a tag file.
